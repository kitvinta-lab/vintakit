# VINTAKIT — Vinta Store

متجر إلكتروني (Landing page) لبيع الملابس الفينتاج، مبني بصفحة واحدة (HTML/CSS/JS خالص، بدون أي framework أو build step).

## 🗂️ هيكل المشروع

```
vintakit/
├── index.html      # الصفحة الرئيسية (البنية HTML)
├── css/
│   └── style.css   # كل التنسيقات (Styles)
├── js/
│   └── script.js   # منطق الموقع (المنتجات، السلة، لوحة التحكم...)
└── README.md
```

## ✨ المزايا

- عرض المنتجات مع فلاتر (المنطقة، المقاسات...)
- صفحة تفاصيل لكل منتج مع معرض صور
- إرسال الطلب مباشرة عبر واتساب (WhatsApp)
- دعم اللغتين العربية والإنجليزية (RTL/LTR)
- لوحة تحكم إدارية (Admin panel) لإضافة/تعديل/حذف المنتجات، محمية بكلمة مرور
- تخزين البيانات محليًا في المتصفح (localStorage) — لا حاجة لأي خادم (backend) أو قاعدة بيانات

## 🚀 التشغيل محليًا

ما عليك سوى فتح ملف `index.html` مباشرة في المتصفح، أو تشغيل خادم بسيط:

```bash
# باستخدام Python
python3 -m http.server 8000

# ثم افتح المتصفح على
http://localhost:8000
```

## 🌐 النشر على GitHub Pages

1. ارفع هذا المجلد إلى مستودع (repository) على GitHub.
2. من إعدادات المستودع (Settings) → Pages.
3. اختر الفرع (Branch) الرئيسي والمجلد الجذري (root).
4. سيصبح الموقع متاحًا على رابط من نوع:
   `https://<username>.github.io/<repo-name>/`

## ⚙️ الإعداد

- **رقم واتساب**: يمكن تعديله من داخل لوحة التحكم (Admin panel داخل الموقع).
- **كلمة مرور لوحة التحكم الافتراضية**: `vintakit2024` (يُنصح بتغييرها من داخل الكود في `js/script.js` قبل النشر الفعلي).
